# Attendance Manager

An Android app for managing attendance records. Built for ISSATSO.

## Features

- Track attendance records
- Local data storage with SQLite
- Simple and intuitive UI
- SharedPreferences for app settings

## Tech Stack

- Java
- Android Studio
- SQLite Database
- SharedPreferences

## Getting Started

1. Clone the repo
2. Open in Android Studio
3. Build and run on an emulator or device

## Requirements

- Android Studio
- JDK 8 or higher
- Android SDK
